# Prompts (installer source files)

These numbered files are **not a tutorial sequence**. The numbering is installer-internal ordering - the installer reads specific files by name and writes them into the target vault under `.claude/commands/`, `.agent/workflows/`, `.agents/skills/`, etc.

To **use** these prompts, run an installer (`install.sh` / `install.ps1` at the repo root). To **understand the system**, read [`../README.md`](../README.md), [`../CLAUDE.md`](../CLAUDE.md), or [`../AGENTS.md`](../AGENTS.md) - not these files in numerical order.
